### Set the site name

```
$site_name = 'My Site';
```

### Set the site description

```
$site_description = 'My Site Description';
```

### Set the comments to open or closed

```
$comments = 'open';
$comments = 'closed';
```

### Set the avatar off
    
```
$avatar = 'off';
```
